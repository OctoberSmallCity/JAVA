



     1.关系操作符 == 比较什么
	 
	     “关系操作符生成的是一个boolean结果，它们计算的是操作数的值之间的关系”。
		 
		 对于基本数据类型来说,比较的是  值  是否相等;

         对于引用类型来说, 比较的是关联的对象在内存中的 地址 ;




     2.equals比较什么

         源码:
             
             public boolean equals(Object obj){
			  
			     return(this == obj);
             }
           
            比较的是两个对象的引用是否相同,即是否指向同一个对象;


         String的特殊之处-改写了equals方法

                 对于String改写了equals方法,用来比较字符串的值是否相等;(当然也必须改写hashcode方法,要不然会导致两个相同的对象存储下来)


         equals方法不能作用于基本数据类型的变量;
